## 📁 폴더 구조 예시
✅ 폴더 구조는 변경될 수 있습니다
```javascript
src/
├── components/
│   └── Component.tsx         # 컴포넌트
├── scenes/
│   ├── TeamMemberA.tsx       # 예시 씬 컴포넌트
│   └── index.ts              # sceneId로 컴포넌트 매핑
├── pages/
│   ├── Home.tsx              # 씬 선택 페이지
│   └── Viewer.tsx            # 선택한 씬 보여주는 페이지
├── App.tsx                   # 라우팅 설정
└── main.tsx                  # 앱 진입점
```
